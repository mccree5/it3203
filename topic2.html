Topic 2: CSS Grid & Flexbox

CSS offers two powerful layout systems. Grid is best for two-dimensional layouts (rows and columns). Flexbox excels at one-dimensional flow (a row or a column). Use them together for clean, responsive pages.

CSS Grid Overview

  • Great for arranging cards, galleries, and full-page sections.
  • Controls both columns and rows with grid-template-columns, grid-template-rows, and gap.
  • Responsive patterns with repeat(), auto-fit, and minmax() minimize media queries.

Flexbox Overview

  • Ideal for navbars, toolbars, and vertical/horizontal alignment within a single row or column.
  • Key properties include display: flex, justify-content, align-items, and gap.
  • Combine with Grid: use Flex inside cards, or Grid for overall page sections.

When to Use Which?

  • Use Grid for page sections, dashboards, galleries, and card layouts.
  • Use Flex for nav bars, toolbars, buttons in a row, and simple alignment.
  • Combine them: Grid for the big picture, Flex inside components.
